# Tugas 2 Cloud Computing

**Nama:** Maulida Rahmah

## Deskripsi

Website Bakuliner Kalimantan Selatan dibuat menggunakan PHP, MySQL, Docker, dan Docker Compose.

## Cara Menjalankan

1. Buka terminal pada folder project.
2. Jalankan perintah:

```bash
docker compose up --build
```

3. Buka browser dan akses:

```
http://localhost:8080
```

## Teknologi yang Digunakan

- PHP
- MySQL
- Docker
- Docker Compose
